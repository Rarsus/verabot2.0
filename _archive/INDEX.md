## Archive Policy

### Archival Criteria
Documentation is moved to `_archive` when:
1. A phase is completed and summarized
2. Session-specific documentation is finalized
3. Implementation guides become historical reference
4. Completion checklists are no longer active

### Access
All archived documentation remains accessible:
- Files are not deleted, only relocated
- Structure maintains chronological organization
- Search-friendly directory names

### Future Archival
Phase 22.3 documentation will be archived after completion to `/phase-22.3/`
